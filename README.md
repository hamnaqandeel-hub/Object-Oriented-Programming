# Library Management System

## Overview

The Library Management System (LMS) is a Java-based desktop application developed to automate library operations. The system helps manage books, members, publishers, and borrowing records efficiently through a user-friendly graphical interface.

## Features

### User Authentication

* Secure login interface for system access.

### Book Management

* Add and manage book records.
* Store book details such as ID, title, author, price, availability, and publisher information.

### Member Management

* Register and manage library members.
* Store member details including membership and expiry dates.

### Publisher Management

* Maintain publisher information and records.

### Borrowing Management

* Track borrowed books and manage borrowing transactions.

### Search and Tracking

* Easy management and tracking of library resources.

## Technologies Used

* Java
* Java Swing (GUI)
* Object-Oriented Programming (OOP)

## How to Run

1. Open the project in Eclipse or NetBeans.
2. Compile all Java files.
3. Run the `login.java` file.
4. Log in and access the Library Management System dashboard.
